üåê Internet & Computer Basics

What is the Internet?
A global network connecting computers for sharing information.
What is a client-server model?
Client (browser) sends request ‚Üí Server responds with data.
What is a domain?
Human-readable name for a website (example.com).
What is hosting?
A service that stores website files on a server.
Difference between HTTP and HTTPS?
HTTPS = secure (encrypted with SSL/TLS).
What does a browser do?
Interprets HTML, CSS, JS and displays websites.
Command to list files in terminal?
ls
Command to change directory?
cd folderName
Command to create folder?
mkdir folderName

üèó HTML Basics

What is the role of <!DOCTYPE html>?
Declares document type as HTML5.
What are <html>, <head>, <body>?
Root element, metadata, visible content.
What are heading tags?
<h1> to <h6> for titles (h1 = biggest).
Difference between <span> and <div>?
<span> = inline, <div> = block-level.
What tag is used for hyperlinks?
<a href="url">Link</a>
How do you display an image?
<img src="image.jpg" alt="text">
What are lists in HTML?
Ordered <ol>, unordered <ul>, with <li> items.
What are forms used for?
Collecting user input (<form>, <input>, <textarea>, <button>).
How do you make a table?
<table><tr><th>Header</th><td>Data</td></tr></table>

üé® CSS Basics

How to select elements in CSS?
By tag (p), class (.class), ID (#id).
Difference between inline, internal, external CSS?
Inline = inside tag, internal = <style>, external = .css file.
What is the CSS box model?
Content ‚Üí Padding ‚Üí Border ‚Üí Margin.
Difference between relative and absolute positioning?
Relative = relative to itself, Absolute = relative to parent/ancestor.
What is Flexbox used for?
One-dimensional layout (row/column).
What is Grid used for?
Two-dimensional layout (rows + columns).
How do media queries work?
Make responsive designs using @media rules.
Units difference: px, %, em, rem?
px = fixed, % = relative to parent, em = relative to parent font-size, rem = relative to root font-size.

‚ö° JavaScript Basics

What is JavaScript used for?
Adding interactivity and dynamic behavior to web pages.
Difference between let, const, var?
let = block scope, const = constant, var = function scope (old).
What are JavaScript data types?
String, Number, Boolean, Array, Object, Null, Undefined.
How to declare a function?
function name() {} or const func = () => {}
Difference between == and ===?
== compares value, === compares value + type.
What are template literals?
Strings with backticks + ${} ‚Üí Hello ${name}
Example of a for loop?
for (let i=0; i<5; i++) { console.log(i); }
What is DOM?
Document Object Model, tree representation of HTML.
How to select an element in JS?
document.querySelector("h1")
How to change text with JS?
document.querySelector("h1").innerText = "Hi";
What is an event listener?
Method that listens for user actions. Example: btn.addEventListener("click", () => alert("Clicked!"));
What are ES6 features?
Template literals, Arrow functions, Destructuring, Spread/Rest, let/const.
Difference between spread and rest?
Spread expands (...arr), Rest collects (function(...args)).
